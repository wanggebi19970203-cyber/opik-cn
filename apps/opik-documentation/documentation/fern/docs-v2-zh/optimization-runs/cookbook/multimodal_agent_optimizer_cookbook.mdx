## 什么是多模态优化器示例？

SDK 包含一个完整的示例，在驾驶危险数据集上演示优化视觉代理。它展示了如何通过 `ChatPrompt` 传递图像内容部分、对输出评分，以及在 Optimization Studio 中比较试验。

## 为什么在这里使用优化器？

多模态提示词对措辞和输出结构很敏感。运行 HRPO 或 MetaPrompt 帮助您收敛到更安全、更一致的输出，而无需手动重写提示词。

## SDK 示例如何工作

1. `multimodal_example.py` 加载驾驶危险数据集（图像 + 危险标签）。
2. 多模态 `ChatPrompt` 在文本指令旁边插入图像 URL 内容部分。
3. 指标（Levenshtein ratio）对预测的危险文本与预期标签进行评分。
4. HRPO 使用训练分割优化提示词，使用小的验证分割进行排序。
5. 结果在 Opik UI 中显示（优化运行和试验详情）。
