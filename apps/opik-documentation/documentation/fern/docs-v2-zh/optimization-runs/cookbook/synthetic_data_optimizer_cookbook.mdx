## 此示例涵盖的内容

- 从 Opik 跟踪生成合成问答数据集
- 使用 TinyQA（通过 [tinyqabenchmarkpp](https://pypi.org/project/tinyqabenchmarkpp/)）和 TinyQA++ 等变体
- 使用 MetaPrompt 在合成数据上优化提示词
- 在 Opik UI 中查看结果
