FewShotBayesianOptimizer 是一个复杂的提示词优化工具，使用贝叶斯优化技术将示例问题中的相关示例添加到系统提示词中。

<Note>
  当您的主要目标是找到最佳数量和组合的少样本示例（演示）来配合您的主要指令提示词时，`FewShotBayesianOptimizer` 是一个很好的选择，特别适用于**聊天模型**。如果您的任务性能严重依赖于上下文示例的质量和相关性，此优化器是理想的选择。
</Note>

## 工作原理

`FewShotBayesianOptimizer` 使用贝叶斯优化来找到最佳的少样本示例集和数量，以配合您的基础指令提示词用于聊天模型。它使用 [Optuna](https://optuna.org/)（一个超参数优化框架）来指导搜索最佳的少样本示例集和数量。

## 快速开始

您可以按照以下步骤使用 `FewShotBayesianOptimizer` 来优化提示词：

```python maxLines=1000
from opik_optimizer import FewShotBayesianOptimizer
from opik.evaluation.metrics import LevenshteinRatio
from opik_optimizer import datasets, ChatPrompt

# Initialize optimizer
optimizer = FewShotBayesianOptimizer(
    model="openai/gpt-4",
    model_parameters={
        "temperature": 0.1,
        "max_tokens": 5000
    },
)

# Prepare dataset
dataset = datasets.hotpot(count=300)

# Define metric and prompt (see docs for more options)
def levenshtein_ratio(dataset_item, llm_output):
    return LevenshteinRatio().score(reference=dataset_item["answer"], output=llm_output)

prompt = ChatPrompt(
    messages=[
        {"role": "system", "content": "Provide an answer to the question."},
        {"role": "user", "content": "{question}"}
    ]
)

# Run optimization
results = optimizer.optimize_prompt(
    prompt=prompt,
    dataset=dataset,
    metric=levenshtein_ratio,
    n_samples=100
)

# Access results
results.display()
```

<ParamField path="min_examples" type="int" optional={true} default="2">提示词中包含的最少示例数</ParamField>
<ParamField path="max_examples" type="int" optional={true} default="8">提示词中包含的最多示例数</ParamField>
<ParamField path="n_threads" type="int" optional={true} default="8">并行评估的线程数</ParamField>

<ParamField path="prompt" type="ChatPrompt">要优化的提示词</ParamField>

### 模型支持

使用 `FewShotBayesianOptimizer` 时需要考虑两个模型：
- `FewShotBayesianOptimizer.model`：用于生成少样本模板和占位符的模型。
- `ChatPrompt.model`：用于评估提示词的模型。

`model` 参数接受任何 LiteLLM 支持的模型字符串（例如，`"gpt-4o"`、`"azure/gpt-4"`、`"anthropic/claude-3-opus"`、`"gemini/gemini-1.5-pro"`）。您还可以使用 `model_parameters` 参数传入额外的模型参数：

```python
optimizer = FewShotBayesianOptimizer(
    model="anthropic/claude-3-opus-20240229",
    model_parameters={
        "temperature": 0.7,
        "max_tokens": 4096
    }
)
```
