了解 Agent Optimizer 的最新更新。以下每条记录反映了 `main` 分支上 `sdks/opik_optimizer/pyproject.toml` 的版本升级。比较链接直接跳转到该版本中的提交。由于 Opik 是一个 monorepo，你可能会在下面的提交链接中看到其他与优化器无关的变更。

## 版本 3.x

亮点：完整的公开、生产就绪版本，与 Optimization Studio 配合使用。由于兼容性原因，MCP/工具优化在 `3.0.0`–`3.0.1` 中暂时移除。

### 版本 3 亮点

- 首次完全原生的跨优化器端到端多智能体和多提示词优化支持。
- 所有优化器支持与训练分离的专用推理模型。
- 所有优化器支持多模态优化。
- 通过 `n` 参数和选择策略在所有优化器中支持 Pass@k 评估。
- 所有优化器的全面重构和规范化，代码更简洁，性能更优。
