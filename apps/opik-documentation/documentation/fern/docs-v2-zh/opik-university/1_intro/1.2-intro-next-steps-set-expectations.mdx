## Opik 大学的预期内容

本视频为您在 Opik 大学的学习之旅奠定基础。您将了解即将推出的视频系列的结构，学习将涵盖的主题，并发现开始使用 Opik 综合 LLM 平台进行实践所需的内容。

## 关键亮点

- **全面覆盖**：系列详细涵盖[可观测性](/v1/tracing/log_traces)、[评估](/v1/evaluation/overview)、[提示工程](/v1/prompt_engineering/prompt_management)、[测试](/v1/testing/pytest_integration)和[生产监控](/v1/production/production_monitoring)
- **实践学习**：每个视频包含可通过链接访问的示例 notebook，所有内容可在 Opik 文档和[实战手册](/v1/integrations/overview)中找到
- **碎片化格式**：视频专注于特定任务，便于在需要时参考
- **灵活的学习路径**：按顺序观看以构建概念，或根据需要跳转到特定主题
- **简单设置**：需要 Comet 账户（在 https://comet.com 免费注册）和基本的 Python/LLM 熟悉程度
- **框架灵活性**：从广泛的[集成](/v1/integrations/overview)页面中选择，或使用 [LiteLLM](/v1/integrations/litellm)/OpenRouter 处理不支持的模型
- **适合初学者**：高级概念使其即使对有学习意愿的初学者也易于理解
- **完整准备**：系列结束时，您将拥有强大的 LLM [可观测性](/v1/tracing/log_traces)、[评估](/v1/evaluation/overview)和[监控](/v1/production/production_monitoring)所需的所有知识
