## 什么是 Opik？

Opik 是 Comet 的综合平台，用于 LLM [可观测性](/v1/tracing/log_traces)、[评估](/v1/evaluation/overview)和[监控](/v1/production/production_monitoring)，旨在支持您从开发到生产的整个 LLM 生命周期。本视频提供了 Opik 核心功能的完整概述，并展示了它如何改变您构建和维护 LLM 应用的方法。

## 关键亮点

- **完整的 LLM 生命周期支持**：从初始开发到生产部署和[监控](/v1/production/production_monitoring)
- **集中式[追踪](/v1/tracing/log_traces)**：在一个位置跟踪与 LLM 应用的所有交互，轻松识别问题和调试复杂链路
- **自动化[评估](/v1/evaluation/overview)**：使用标准化的[数据集](/v1/evaluation/manage_datasets)和[指标](/v1/evaluation/metrics/overview)替代临时的"感觉检查"，实现一致、可重复的评估
- **[生产监控](/v1/production/production_monitoring)**：使用[测试](/v1/testing/pytest_integration)和监控工具实时跟踪性能，自信部署
- **开源 + 企业版**：获得开源的灵活性以及企业解决方案的安全性、可扩展性和支持
- **丰富的[集成](/v1/integrations/overview)生态系统**：与 [OpenAI](/v1/integrations/openai)、Azure 等流行框架无缝协作
- **全面的文档**：包含[快速入门](/v1/quickstart)指南、[集成](/v1/integrations/overview)和可在 Google Colab 中直接运行的 Jupyter notebook [实战手册](/v1/integrations/overview)的丰富文档
