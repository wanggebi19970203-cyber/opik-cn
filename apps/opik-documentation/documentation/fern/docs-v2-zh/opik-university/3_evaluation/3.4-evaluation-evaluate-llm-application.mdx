## 整合一切：完整的 LLM 评估

本综合视频演示 Opik 中完整的[评估](/v1/evaluation/overview)工作流，其中[数据集](/v1/evaluation/manage_datasets)和[指标](/v1/evaluation/metrics/overview)结合在一起系统地评估 LLM 性能。您将看到 GPT-4 和 [Gemini](/v1/integrations/gemini) 模型在 RAG 应用上的实际比较，了解[提示版本控制](/v1/prompt_engineering/prompt_management)、[实验](/v1/evaluation/evaluate_your_llm)管理，并发现如何为生产部署做出数据驱动的决策。这是所有先前概念统一为可操作见解的地方。

## 关键亮点

- **端到端[评估](/v1/evaluation/overview)工作流**：运行完整的[评估](/v1/evaluation/evaluate_your_llm)，在系统化管道中处理[数据集](/v1/evaluation/manage_datasets)、应用模型并使用定义的[指标](/v1/evaluation/metrics/overview)对输出进行评分
- **[提示管理](/v1/prompt_engineering/prompt_management)和版本控制**：使用 Opik 的[提示类](/v1/prompt_engineering/prompt_management)创建带有提交历史的版本化[提示](/v1/prompt_engineering/prompt_management)，确保可重复性并节省时间/金钱
- **多模型基准测试**：使用[评估任务](/v1/evaluation/evaluate_your_llm)和系统化评分在相同[数据集](/v1/evaluation/manage_datasets)上并排比较不同模型（GPT-4 vs [Gemini](/v1/integrations/gemini)）
- **智能[实验](/v1/evaluation/evaluate_your_llm)组织**：战略性地命名[实验](/v1/evaluation/evaluate_your_llm)（例如按模型名称）以便于识别和比较，而不是依赖随机生成的名称
- **实时[实验](/v1/evaluation/evaluate_your_llm)监控**：通过 Opik UI 实时跟踪[评估](/v1/evaluation/overview)进度，查看[数据集](/v1/evaluation/manage_datasets)处理和结果生成
- **并排比较**：使用比较功能同时评估多个[实验](/v1/evaluation/evaluate_your_llm)，基于定量[指标](/v1/evaluation/metrics/overview)做出模型选择决策
- **模板生成**：利用"创建新实验"按钮自动生成带有选定[指标](/v1/evaluation/metrics/overview)的[评估](/v1/evaluation/overview)脚本，以便在 Python 中重用。模态框中的每个指标都包含文档链接以供快速参考
- **追踪级检查**：通过从[实验](/v1/evaluation/evaluate_your_llm)结果打开[追踪](/v1/tracing/log_traces)深入查看单个响应，了解模型行为和决策路径
- **数据驱动的生产决策**：基于具体[指标](/v1/evaluation/metrics/overview)选择最佳性能的[提示](/v1/prompt_engineering/prompt_management)和模型，而不是主观评估，为部署建立信心
