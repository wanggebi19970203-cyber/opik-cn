# 为 Agent 优化器 SDK 做贡献

本指南将帮助你开始为 Agent 优化器 SDK 做出贡献，这是我们用于优化提示词和提升模型性能的工具。

在开始之前，请查看我们的[贡献概述](/contributing/overview)和[贡献者许可协议 (CLA)](https://github.com/comet-ml/opik/blob/main/CLA.md)。

## 项目结构

Agent 优化器位于 `sdks/opik_optimizer` 目录中。以下是关键组件的概述：

- `src/`：主要源代码
- `benchmarks/`：基准测试工具和结果
- `notebooks/`：示例笔记本和教程
- `tests/`：测试文件
- `docs/`：附加文档
- `scripts/`：实用脚本
- `setup.py`：包配置
- `requirements.txt`：Python 依赖项

## 环境设置

<Steps>
  <Step title="创建虚拟环境">
    ```bash
    python -m venv venv
    source venv/bin/activate  # Windows 系统: venv\Scripts\activate
    ```
  </Step>
  <Step title="安装依赖项">
    ```bash
    cd sdks/opik_optimizer
    pip install -r requirements.txt
    pip install -e .
    ```
  </Step>
  <Step title="运行测试">
    ```bash
    pytest tests/
    ```
  </Step>
</Steps>
