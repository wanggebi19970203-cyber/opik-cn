[Agno](https://github.com/agno-agi/agno) 是一个轻量级、高性能的 AI 代理构建库。

Agno 的主要优势在于其极低的开销和高效的执行能力，使其成为对性能要求严格的生产环境的理想选择，同时保持代理开发的简洁性。

## 快速开始

要使用 Agno 与 Opik 的集成，您需要安装以下依赖包：

```bash
pip install -U agno openai opentelemetry-sdk opentelemetry-exporter-otlp openinference-instrumentation-agno yfinance
```

此外，您需要设置以下环境变量来配置 OpenTelemetry 集成：

<Tabs>
    <Tab value="Opik Cloud" title="Opik Cloud">
        如果您使用 Opik Cloud，需要设置以下环境变量：
            
        ```bash wordWrap
        export OTEL_EXPORTER_OTLP_ENDPOINT=https://www.comet.com/opik/api/v1/private/otel
        export OTEL_EXPORTER_OTLP_HEADERS='Authorization=<your-api-key>,Comet-Workspace=default'
        ```

        <Tip>
            要将追踪记录到特定项目，可以在 `OTEL_EXPORTER_OTLP_HEADERS`
            环境变量中添加 `projectName` 参数：

            ```bash wordWrap
            export OTEL_EXPORTER_OTLP_HEADERS='Authorization=<your-api-key>,Comet-Workspace=default,projectName=<your-project-name>'
            ```

            如果需要将数据记录到不同的工作区，也可以将 `Comet-Workspace` 参数更新为其他值。
        </Tip>
    </Tab>
    <Tab value="Enterprise deployment" title="企业部署">
        如果您使用 Opik 企业部署版本，需要设置以下环境变量：

        ```bash wordWrap
        export OTEL_EXPORTER_OTLP_ENDPOINT=https://<comet-deployment-url>/opik/api/v1/private/otel
        export OTEL_EXPORTER_OTLP_HEADERS='Authorization=<your-api-key>,Comet-Workspace=default'
        ```

        <Tip>
            要将追踪记录到特定项目，可以在 `OTEL_EXPORTER_OTLP_HEADERS`
            环境变量中添加 `projectName` 参数：

            ```bash wordWrap
            export OTEL_EXPORTER_OTLP_HEADERS='Authorization=<your-api-key>,Comet-Workspace=default,projectName=<your-project-name>'
            ```

            如果需要将数据记录到不同的工作区，也可以将 `Comet-Workspace` 参数更新为其他值。
        </Tip>
    </Tab>
    <Tab value="Self-hosted instance" title="自托管实例">

    如果您使用自托管的 Opik，需要设置以下环境变量：

    ```bash
    export OTEL_EXPORTER_OTLP_ENDPOINT=http://localhost:5173/api/v1/private/otel

        要将追踪记录到特定项目，可以在 `OTEL_EXPORTER_OTLP_HEADERS`

        export OTEL_EXPORTER_OTLP_HEADERS='projectName=<your-project-name>'
        ```

## 在 Agno 中使用 Opik

以下示例展示了如何在 Agno 中使用 Opik 集成：

```python
from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.yfinance import YFinanceTools
from openinference.instrumentation.agno import AgnoInstrumentor
from opentelemetry import trace as trace_api
from opentelemetry.exporter.otlp.proto.http.trace_exporter import OTLPSpanExporter
from opentelemetry.sdk.trace import TracerProvider
from opentelemetry.sdk.trace.export import SimpleSpanProcessor

# 配置追踪器提供者
tracer_provider = TracerProvider()
tracer_provider.add_span_processor(SimpleSpanProcessor(OTLPSpanExporter()))
trace_api.set_tracer_provider(tracer_provider=tracer_provider)

# 开始对 agno 进行插桩
AgnoInstrumentor().instrument()

# 创建并配置代理
agent = Agent(
    name="Stock Price Agent",
    model=OpenAIChat(id="gpt-4o-mini"),
    tools=[YFinanceTools()],
    instructions="You are a stock price agent. Answer questions in the style of a stock analyst.",
    debug_mode=True,
)

# 使用代理
agent.print_response("What is the current price of Apple?")
```
