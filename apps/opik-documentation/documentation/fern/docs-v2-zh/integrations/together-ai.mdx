[Together AI](https://www.together.ai/) 为领先的开源模型提供快速推理服务，包括 Llama、Mistral、Qwen 等。

本指南介绍如何通过 LiteLLM 将 Opik 与 Together AI 集成。通过使用 Opik 提供的 LiteLLM 集成，您可以轻松地在 Opik 项目中跟踪和评估 Together AI 调用，Opik 会自动记录输入提示词、使用的模型、token 用量和生成的响应。

## 快速开始

### 配置 Opik

要开始跟踪 Together AI 调用，您需要安装 `opik` 和 `litellm`。可以使用 pip 进行安装：

```bash
pip install opik litellm
```

此外，您可以使用 `opik configure` 命令配置 Opik，该命令会提示您输入正确的本地服务器地址，或者如果您使用云平台，则需要输入 API 密钥：

```bash
opik configure
```

### 配置 Together AI

您需要将 Together AI API 密钥设置为环境变量：

```bash
export TOGETHER_API_KEY="YOUR_API_KEY"
```

## 记录 LLM 调用

要将 LLM 调用记录到 Opik，您需要创建 OpikLogger 回调函数。创建并添加 OpikLogger 回调到 LiteLLM 后，您就可以像往常一样调用 LiteLLM：

```python
from litellm.integrations.opik.opik import OpikLogger
import litellm

opik_logger = OpikLogger()
litellm.callbacks = [opik_logger]

response = litellm.completion(
    model="together_ai/meta-llama/Llama-3.2-3B-Instruct-Turbo",
    messages=[
        {"role": "user", "content": "Why is tracking and evaluation of LLMs important?"}
    ]
)
```

## 在被跟踪的函数中记录 LLM 调用

如果您在使用 [`@track`](/tracing/advanced/log_traces#using-function-decorators) 装饰器跟踪的函数中使用 LiteLLM，您需要将 `current_span_data` 作为元数据传递给 `litellm.completion` 调用：

```python
from opik import track, opik_context
import litellm

@track
def generate_story(prompt):
    response = litellm.completion(
        model="together_ai/meta-llama/Llama-3.2-3B-Instruct-Turbo",
        messages=[{"role": "user", "content": prompt}],
        metadata={
            "opik": {
                "current_span_data": opik_context.get_current_span_data(),
            },
        },
    )
    return response.choices[0].message.content


@track
def generate_topic():
    prompt = "Generate a topic for a story about Opik."
    response = litellm.completion(
        model="together_ai/meta-llama/Llama-3.2-90B-Vision-Instruct-Turbo",
        messages=[{"role": "user", "content": prompt}],
        metadata={
            "opik": {
                "current_span_data": opik_context.get_current_span_data(),
            },
        },
    )
    return response.choices[0].message.content


@track
def generate_opik_story():
    topic = generate_topic()
    story = generate_story(topic)
    return story


generate_opik_story()
```
