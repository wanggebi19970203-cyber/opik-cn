[Fireworks AI](https://fireworks.ai/) 为流行的开源模型提供快速推理服务，通过优化的推理基础设施，提供对 Llama、Mistral 和 Qwen 等模型的高性能 API 访问。

本指南介绍如何通过 Fireworks AI 的 OpenAI 兼容 API 端点将 Opik 与 Fireworks AI 集成。通过使用 Opik OpenAI 集成，您可以轻松地在 Opik 项目中跟踪和评估 Fireworks AI API 调用，Opik 会自动记录输入提示词、使用的模型、token 用量和生成的响应。

![Fireworks AI 集成结果](/img/tracing/fireworks_ai_integration.png)

## 快速开始

### 账户设置

首先，您需要一个 Comet.com 账户才能使用 Opik。如果还没有账户，可以[免费注册](https://www.comet.com/signup?utm_source=opik&utm_medium=docs&utm_campaign=fireworks-ai-integration)。

### 安装

安装所需的包：

```bash
pip install opik openai
```

### 配置

配置 Opik 将跟踪数据发送到您的 Comet 项目：

```python
import opik

opik.configure(
    project_name="your-project-name",
    workspace="your-workspace-name",
)
```

### 环境设置

将 Fireworks AI API 密钥设置为环境变量：

```bash
export FIREWORKS_API_KEY="your-fireworks-api-key"
```

您可以从 [Fireworks AI 控制台](https://fireworks.ai/api-keys) 获取 API 密钥。

## 基本用法

### 导入和设置

```python
from opik.integrations.openai import track_openai
from openai import OpenAI

# 创建 OpenAI 客户端
client = OpenAI(
    api_key="your-fireworks-api-key",
    base_url="https://api.fireworks.ai/inference/v1"
)

# 使用 Opik 跟踪包装客户端
tracked_client = track_openai(client, project_name="your-project-name")
```

### 发起 API 调用

```python
# 发起聊天补全请求
response = tracked_client.chat.completions.create(
    model="accounts/fireworks/models/llama-v3p1-8b-instruct",
    messages=[
        {"role": "system", "content": "You are a helpful assistant."},
        {"role": "user", "content": "What are the benefits of using fast inference platforms?"}
    ]
)

print(response.choices[0].message.content)
```

## 高级用法

### 结合 @track 装饰器使用

您可以将跟踪客户端与 Opik 的 `@track` 装饰器结合使用，实现更全面的跟踪：

```python
from opik import track

@track
def analyze_text_with_fireworks(text: str):
    response = tracked_client.chat.completions.create(
        model="accounts/fireworks/models/llama-v3p1-8b-instruct",
        messages=[
            {"role": "user", "content": f"Analyze this text: {text}"}
        ],
    )
    return response.choices[0].message.content

# 使用该函数
result = analyze_text_with_fireworks("Open source AI models are becoming increasingly powerful.")
```

![使用 @track 装饰器的 Fireworks AI 函数跟踪](/img/tracing/fireworks_ai_integration_2.png)

### 流式响应

Fireworks AI 支持流式响应，Opik 同样会对其进行跟踪：

```python
response = tracked_client.chat.completions.create(
    model="accounts/fireworks/models/llama-v3p1-8b-instruct",
    messages=[
        {"role": "user", "content": "Explain quantum computing in simple terms."}
    ],
    stream=True
)

for chunk in response:
    if chunk.choices[0].delta.content is not None:
        print(chunk.choices[0].delta.content, end="")
```

## 支持的模型

Fireworks AI 为多种流行的开源模型提供快速推理服务。可用的热门模型包括：

- **Llama 系列**: `accounts/fireworks/models/llama-v3p1-8b-instruct`, `accounts/fireworks/models/llama-v3-70b-instruct`
- **Mistral 系列**: `accounts/fireworks/models/mixtral-8x7b-instruct-hf`, `accounts/fireworks/models/mistral-7b-instruct-v0.1`
- **Qwen 系列**: `accounts/fireworks/models/qwen-72b-chat`, `accounts/fireworks/models/qwen-14b-chat`
- **代码模型**: `accounts/fireworks/models/starcoder-16b`, `accounts/fireworks/models/codellama-34b-instruct-hf`

完整的可用模型列表，请访问 [Fireworks AI 模型目录](https://fireworks.ai/models)。

## 查看结果

当您的 Fireworks AI 调用通过 Opik 记录后，您可以在 Opik 仪表板中查看详细的跟踪信息。每次 API 调用都会创建一个跟踪记录，包含：

- 输入提示词和消息
- 模型参数和配置
- 响应内容和元数据
- token 用量统计
- 计时信息
- 您添加的任何自定义元数据

## 故障排除

### 常见问题

**API 密钥问题**：确保您的 `FIREWORKS_API_KEY` 环境变量已正确设置且有足够的额度。

**模型名称格式**：Fireworks AI 模型使用 `accounts/fireworks/models/{model-name}` 格式。请确保使用正确的模型标识符。

**速率限制**：Fireworks AI 根据您的套餐设有速率限制。如果遇到速率限制，建议在应用中实现指数退避重试。

**Base URL**：Fireworks AI 的 Base URL 为 `https://api.fireworks.ai/inference/v1`。请确保使用正确的端点。

### 获取帮助

如果在集成过程中遇到问题：

1. 检查您的 API 密钥和模型名称
2. 验证您的 Opik 配置
3. 查阅 [Fireworks AI 文档](https://docs.fireworks.ai/)
4. 查看 [Opik OpenAI 集成文档](/integrations/openai)
