## 快速开始

### 安装

要开始跟踪您的 Groq LLM 调用，您可以使用我们的 [LiteLLM 集成](/integrations/litellm)。您需要安装 `opik` 和 `litellm` 两个包。可以使用 pip 安装：

```bash
pip install opik litellm
```

### 配置 Opik

根据您的部署类型配置 Opik Python SDK。有关详细说明，请参阅 [Python SDK 配置指南](/tracing/advanced/sdk_configuration)：

- **CLI 配置**：`opik configure`
- **代码配置**：`opik.configure()`
- **自托管 vs 云托管 vs 企业版**设置
- **配置文件**和环境变量

<Info>

如果您无法使用我们的 LiteLLM 集成连接 Groq，请[提交 issue](https://github.com/comet-ml/opik/issues/new/choose)

</Info>

### 配置 Groq

要配置 Groq，您需要拥有 Groq API Key。您可以在[此页面](https://console.groq.com/keys)创建和管理您的 Groq API Key。

可以将其设置为环境变量：

```bash
export GROQ_API_KEY="YOUR_API_KEY"
```

或在代码中动态设置：

```python
import os
import getpass

if "GROQ_API_KEY" not in os.environ:
    os.environ["GROQ_API_KEY"] = getpass.getpass("Enter your Groq API key: ")
```

## 记录 LLM 调用

要将 LLM 调用记录到 Opik，您需要创建 OpikLogger 回调。创建 OpikLogger 回调并添加到 LiteLLM 后，您就可以像往常一样调用 LiteLLM：

```python
from litellm.integrations.opik.opik import OpikLogger
import litellm
import os

os.environ["OPIK_PROJECT_NAME"] = "groq-integration-demo"

opik_logger = OpikLogger()
litellm.callbacks = [opik_logger]

prompt = """
Write a short two sentence story about Opik.
"""

response = litellm.completion(
    model="groq/llama3-8b-8192",
    messages=[{"role": "user", "content": prompt}]
)

print(response.choices[0].message.content)
```

## 高级用法

### 与 `@track` 装饰器配合使用

如果您在使用 [`@track`](/tracing/advanced/log_traces#using-function-decorators) 装饰器跟踪的函数中使用 LiteLLM，需要将 `current_span_data` 作为元数据传递给 `litellm.completion` 调用：

```python
from opik import track
from opik.opik_context import get_current_span_data
import litellm

@track
def generate_story(prompt):
    response = litellm.completion(
        model="groq/llama3-8b-8192",
        messages=[{"role": "user", "content": prompt}],
        metadata={
            "opik": {
                "current_span_data": get_current_span_data(),
            },
        },
    )
    return response.choices[0].message.content

@track
def generate_topic():
    prompt = "Generate a topic for a story about Opik."
    response = litellm.completion(
        model="groq/llama3-8b-8192",
        messages=[{"role": "user", "content": prompt}],
        metadata={
            "opik": {
                "current_span_data": get_current_span_data(),
            },
        },
    )
    return response.choices[0].message.content

@track
def generate_opik_story():
    topic = generate_topic()
    story = generate_story(topic)
    return story

# 执行多步骤流水线
generate_opik_story()
```
