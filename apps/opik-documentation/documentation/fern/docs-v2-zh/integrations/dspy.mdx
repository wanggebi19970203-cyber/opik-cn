[DSPy](https://dspy.ai/) 是一个用于编程而非提示语言模型的框架。

在本指南中，我们将展示如何将 Opik 与 DSPy 集成，以便所有 DSPy 调用都作为跟踪记录在 Opik 中。

## 账户设置

[Comet](https://www.comet.com/site?from=llm&utm_source=opik&utm_medium=colab&utm_content=dspy&utm_campaign=opik) 提供托管版本的 Opik 平台，[只需创建账户](https://www.comet.com/signup?from=llm&utm_source=opik&utm_medium=colab&utm_content=dspy&utm_campaign=opik) 并获取你的 API 密钥。

## 快速开始

### 安装

首先，确保你已安装 `opik` 和 `dspy`：

```bash
pip install opik dspy
```

### 配置 Opik

根据你的部署类型配置 Opik Python SDK。请参阅 [Python SDK 配置指南](/tracing/advanced/sdk_configuration) 了解以下详细信息：

- **CLI 配置**：`opik configure`
- **代码配置**：`opik.configure()`
- **自托管 vs 云 vs 企业版** 设置
- **配置文件** 和环境变量

### 配置 DSPy

要配置 DSPy，你需要有 LLM 提供商的 API 密钥。本示例使用 OpenAI。你可以在[此页面查找或创建 OpenAI API 密钥](https://platform.openai.com/settings/organization/api-keys)。

你可以将其设置为环境变量：

```bash
export OPENAI_API_KEY="YOUR_API_KEY"
```

或者在程序中设置：

```python
import os
import getpass

if "OPENAI_API_KEY" not in os.environ:
    os.environ["OPENAI_API_KEY"] = getpass.getpass("Enter your OpenAI API key: ")
```

## 记录 DSPy 调用

要将跟踪记录发送到 Opik，你需要设置 `opik` 回调：

```python
import dspy
from opik.integrations.dspy.callback import OpikCallback

lm = dspy.LM("openai/gpt-4o-mini")

project_name = "DSPY"
opik_callback = OpikCallback(project_name=project_name, log_graph=True)

dspy.configure(lm=lm, callbacks=[opik_callback])

cot = dspy.ChainOfThought("question -> answer")
cot(question="What is the meaning of life?")
```
