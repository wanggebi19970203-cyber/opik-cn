管理仪表板中的用户部分允许组织管理员查看所有用户、更改组织角色以及从组织中移除用户。

## 查看用户

要查看组织中的所有用户：

1. 导航到[管理仪表板](/administration/admin-dashboard/overview)
2. 在侧边栏中点击 **Users**

列表显示每个用户的姓名、电子邮件、组织角色以及加入时间。

## 更改组织角色

组织角色决定组织范围内的高级访问权限。有关可用角色的详细信息，请参阅[角色和权限](/administration/roles_and_permissions)。

要更改用户的组织角色：

1. 导航到管理仪表板中的 **Users** 部分
2. 找到要更改角色的用户
3. 点击角色下拉菜单
4. 选择新角色

<Warning>
您无法更改自己的组织角色。必须由其他管理员来更改您的角色。组织中必须始终至少有一名管理员。
</Warning>

## 移除用户

要从组织中移除用户：

1. 导航到管理仪表板中的 **Users** 部分
2. 找到要移除的用户
3. 点击 **更多选项** 菜单（三个点）
4. 选择 **从组织中移除**
5. 确认移除

<Warning>
移除用户会立即撤销其所有访问权限。其历史数据（跟踪、实验）将被保留。您无法移除自己或组织中的最后一名管理员。
</Warning>
