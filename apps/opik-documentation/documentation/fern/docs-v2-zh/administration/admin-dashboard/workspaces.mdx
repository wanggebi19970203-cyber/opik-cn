工作空间将组织内的相关项目分组在一起。每个工作空间拥有独立的成员、角色和隔离的数据（追踪、实验、数据集）。

## 查看工作空间

要查看组织中的所有工作空间：

1. 导航至[管理控制台](/administration/admin-dashboard/overview)
2. 点击侧边栏中的 **工作空间**

列表中会显示每个工作空间的名称、成员数量和创建日期。

## 创建工作空间

1. 导航至管理控制台中的 **工作空间** 部分
2. 点击 **创建工作空间**
3. 输入工作空间的**名称**
4. 点击 **创建**

<Tip>
使用清晰、具有描述性的名称。例如：`production`、`staging`、`research-team`。
</Tip>

## 删除工作空间

<Warning>
删除工作空间会永久移除其中的所有数据，包括项目、追踪、实验和数据集。此操作不可撤销。
</Warning>

要删除工作空间：

1. 导航至管理控制台中的 **工作空间** 部分
2. 找到您要删除的工作空间
3. 点击**更多选项**菜单（三个点）
4. 选择 **删除**
5. 输入工作空间名称以确认删除

您无法删除组织中唯一的工作空间。
