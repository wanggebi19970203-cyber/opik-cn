<Note>
组织设置功能适用于 Opik Cloud 和企业版计划。此功能在开源部署中不可用。如果您希望为您的 Opik 部署启用此功能，请[联系我们](https://www.comet.com/site/about-us/contact-us/)。
</Note>

组织设置允许管理员配置适用于整个组织的设置。

## 访问组织设置

1. 导航至[管理控制台](/administration/admin-dashboard/overview)
2. 点击侧边栏中的**通用设置**

## 限制用户邀请仅限管理员

默认情况下，组织中的任何成员都可以邀请新用户，这可能会影响计费。启用此设置可将邀请限制为仅限组织管理员。

启用后：
- 仅组织管理员可以邀请用户加入组织
- 普通成员无法发送邀请

禁用时：
- 所有成员都可以邀请用户加入组织
