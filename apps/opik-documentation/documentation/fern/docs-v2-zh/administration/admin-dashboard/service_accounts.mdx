服务账户为自动化系统、CI/CD 流水线和集成提供对 Opik 的编程式访问。与用户账户不同，服务账户使用 API 密钥进行身份验证，无需交互式登录。

<Note>
服务账户功能适用于 Opik Cloud 和 Enterprise 计划。此功能在开源部署中不可用。如果您希望为您的 Opik 部署启用此功能，请[联系我们](https://www.comet.com/site/about-us/contact-us/)。
</Note>

## 创建服务账户

1. 导航到[管理控制台](/administration/admin-dashboard/overview)
2. 在侧边栏中点击**服务账户**
3. 点击**创建服务账户**
4. 输入服务账户的**名称**
5. 可选择**默认工作区**
6. 可选择**授权工作区**以限制访问
7. 点击**创建**

<Tip>
使用描述性名称来表明服务账户的用途，例如 `ci-pipeline-prod` 或 `monitoring-service`。
</Tip>

## 管理 API 密钥

每个服务账户可以拥有多个 API 密钥。要管理密钥：

1. 在管理控制台中导航到**服务账户**部分
2. 找到目标服务账户并点击**管理 API 密钥**

在 API 密钥弹窗中，您可以：

- **添加密钥**：生成新的 API 密钥
- **复制密钥**：将现有密钥复制到剪贴板
- **删除密钥**：删除密钥（立即生效）

<Warning>
API 密钥仅在创建时显示一次。请立即复制并安全存储。如果丢失密钥，您需要生成新的密钥。
</Warning>

## 删除服务账户

<Warning>
删除服务账户会立即撤销其所有 API 密钥。使用这些密钥的任何系统将停止工作。
</Warning>

1. 在管理控制台中导航到**服务账户**部分
2. 找到要删除的服务账户
3. 点击**删除**操作
4. 确认删除

## 使用服务账户 API 密钥

使用服务账户 API 密钥配置 Opik SDK：

```python
import opik

# Using environment variables (recommended)
# export OPIK_API_KEY=your-service-account-api-key
# export OPIK_WORKSPACE=your-workspace
client = opik.Opik()

# Or configure directly
client = opik.Opik(
    api_key="your-service-account-api-key",
    workspace="your-workspace"
)
```
