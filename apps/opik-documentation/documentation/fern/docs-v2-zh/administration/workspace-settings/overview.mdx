工作区设置允许您配置应用于整个工作区的选项。这些设置与组织级别的管理功能分开管理。

## 访问工作区设置

1. 在 Opik 中导航到您的工作区
2. 点击左侧边栏中的 **配置**
3. 选择要配置的设置对应的选项卡

<Frame>
  <img src="/img/administration/workspace_configuration.png" alt="工作区配置" />
</Frame>

## 谁可以访问工作区设置？

- **工作区所有者**可以访问所有设置，包括成员管理和偏好设置
- 具有相应角色的**工作区成员**可以访问反馈定义和 AI 提供商

<Note>
工作区成员功能适用于 Opik Cloud 和企业版计划。此功能在开源部署中不可用。如果您想为您的 Opik 部署启用此功能，请[联系我们](https://www.comet.com/site/about-us/contact-us/)。
</Note>
