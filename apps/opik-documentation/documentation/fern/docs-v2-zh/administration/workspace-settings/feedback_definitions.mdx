反馈定义允许您创建自定义标签，以便系统地收集和分析 LLM 输出的结构化反馈。

## 使用反馈定义

创建后，反馈定义会出现在两个地方：

1. **跟踪和线程侧边栏** — 在审查单个项目时添加评分。
2. **标注队列** — 在创建队列时选择标注人员可以看到的定义。

<Frame>
  <img src="/img/administration/annotation_queue.png" alt="标注队列反馈选择" />
</Frame>

<Tip>
您也可以使用 SDK 以编程方式记录反馈评分。详见 [SDK 文档](/reference/overview)。
</Tip>

## 创建反馈定义

<Frame>
  <img src="/img/configuration/new_feedback_definition.png" alt="新建反馈定义弹窗" />
</Frame>

1. 点击 **创建新反馈定义**
2. 输入反馈定义的**名称**
3. 选择**类型**：分类型或数值型
4. 定义**取值**（分类型）或**范围**（数值型）

## 常见反馈类型

| 反馈类型 | 类型 | 取值 |
| --- | --- | --- |
| 赞/踩 | 分型 | 赞, 踩 |
| 有用性 | 分型 | 有用, 一般, 无用 |
| 幻觉 | 分型 | 是, 否 |
| 正确性 | 分型 | 好, 差 |
