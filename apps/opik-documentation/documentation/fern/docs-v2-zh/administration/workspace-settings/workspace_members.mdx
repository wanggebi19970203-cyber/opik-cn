<Note>
工作区成员功能适用于 Opik Cloud 和企业版计划。开源部署不支持此功能。如果您希望为 Opik 部署启用此功能，请[联系我们](https://www.comet.com/site/about-us/contact-us/)。
</Note>

## 概述

工作区成员管理允许工作区所有者查看成员、邀请新用户、移除用户和分配工作区角色。

## 访问工作区成员

1. 在 Opik 中导航到您的工作区
2. 点击左侧边栏中的**配置**
3. 选择**成员**选项卡

<Note>
成员选项卡仅对工作区所有者可见。
</Note>

## 查看工作区成员

成员表格显示以下信息：

| 列 | 描述 |
| --- | --- |
| **姓名/用户名** | 成员的显示名称或用户名 |
| **邮箱** | 成员的电子邮箱地址 |
| **加入时间** | 成员加入工作区的时间 |
| **工作区角色** | 成员在此工作区中的角色 |

## 添加用户

邀请新用户：

1. 点击右上角的**添加用户**按钮
2. 通过用户名或电子邮箱搜索用户
3. 选择要添加的用户
4. 点击确认邀请

**添加现有组织成员：**
- 通过用户名搜索已存在于组织中的用户
- 他们将立即被添加到工作区

<Note>
用户名搜索仅返回已属于您组织成员的用户。
</Note>

**通过邮箱邀请：**
- 输入尚未加入组织的人员的电子邮箱地址
- 他们将收到一封邀请邮件
- 一旦他们接受邀请并加入组织，就会被添加到工作区

<Tip>
通过邮箱邀请的用户在接受邀请并创建账户之前，将显示其电子邮箱地址。
</Tip>

## 更改成员角色

1. 在表格中找到该成员
2. 点击**工作区角色**列中的当前角色
3. 从下拉菜单中选择新角色
4. 更改立即生效

<Warning>
组织管理员始终拥有对所有工作区的完全访问权限，无论其分配的工作区角色如何。
</Warning>

有关可用角色和权限的详细信息，请参阅[角色和权限](/administration/roles_and_permissions)。

## 移除成员

从工作区移除成员：

1. 在表格中找到该成员
2. 点击其行上的操作菜单（三个点）
3. 选择**从工作区移除**
4. 确认移除

<Warning>
被移除的用户将立即失去访问权限。其历史数据（跟踪记录、实验）将被保留。您无法将自己从工作区中移除。
</Warning>
