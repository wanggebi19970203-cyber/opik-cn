仪表板允许你创建可自定义的视图来监控 LLM 应用。你可以跟踪项目指标，如 trace 数量、成本、延迟和反馈分数，以及比较不同运行的实验结果。

Opik 提供两种数据可视化方式：

- **Insights** — 直接嵌入项目和实验页面的内置和自定义视图，用于快速的上下文监控
- **工作区仪表板** — 可从侧边栏访问的独立仪表板，用于跨项目分析

如果你对仪表板有任何反馈或功能请求，请[在 GitHub 上提交 issue](https://github.com/comet-ml/opik/issues)。

## 仪表板类型

每个仪表板都有一个**类型**，决定它处理的数据种类和可用的组件：

| 类型 | 用途 | 可用组件 |
|------|---------|-------------------|
| **多项目** | 跨一个或多个项目跟踪指标（trace、线程、成本、延迟） | 时间序列、单一指标、Markdown |
| **实验** | 比较实验运行的反馈分数和结果 | 指标、排行榜、Markdown |

## 访问仪表板

### 仪表板页面

从侧边栏导航访问独立的仪表板页面，创建和管理工作区级仪表板。仪表板列表包含一个**类型**列，显示每个仪表板是多项目还是实验类型。

### 项目页面 — Insights 标签

在任何项目中，**Insights** 标签提供内置和自定义视图来监控该项目的 trace、线程和质量指标。

### 比较实验 — Insights 标签

比较实验时，**Insights** 标签显示一个内置的只读视图，包含实验比较图表。

## Insights 标签

Insights 标签提供精选的、上下文相关的监控视图，直接嵌入项目和实验页面中。

### 项目 Insights

当你打开项目的 Insights 标签时，你会看到内置的**项目概览**视图 — 一个只读仪表板，涵盖关键健康指标：trace 数量、错误、延迟、成本、反馈分数和线程活动。

#### 自定义视图

在内置视图之外，你可以为项目创建自定义 Insights 视图：

1. 在 Insights 标签中打开**视图选择器**下拉菜单
2. 点击底部的**添加新视图**
3. 输入视图名称

自定义视图完全可编辑 — 你可以添加部分、配置组件并重新排列布局。当前项目自动设置为所有组件的数据源。

**视图选择器下拉菜单：**

- 顶部搜索框用于过滤视图
- 内置的"项目概览"始终排在第一位，带有"内置"标签
- 自定义视图显示在其下方，包含组件数量和最后修改日期
- 底部的**添加新视图**按钮

<Frame>
  <img src="/img/production/dashboard_insights_views_selector.png" alt="视图选择器下拉菜单" />
</Frame>

**视图操作**（悬停在自定义视图上时可用）：

- **编辑名称** — 重命名视图
- **复制** — 创建视图的副本
- **删除** — 移除视图（此操作不可撤销）

你还可以**复制内置视图**来创建一个可编辑的副本作为自定义视图。

Insights 标签有自己的时间范围选择器，与日志标签分开。每个标签在会话间记住自己的时间范围。

### 实验 Insights

比较实验时，Insights 标签显示一个单一的内置只读视图，展示当前选定实验的实验比较图表。没有视图选择器 — 只有内置视图可用。

## 组件类型

仪表板支持多种组件类型。可用类型取决于仪表板类型（多项目或实验）。

### 时间序列组件（多项目）

显示项目指标随时间变化的时间序列图表。支持折线图和柱状图可视化。

**可用指标：**

- **Trace 反馈分数** — trace 的质量指标随时间变化
- **Trace 数量** — trace 数量趋势
- **Trace 持续时间** — trace 性能趋势
- **Token 使用量** — token 消耗随时间变化
- **预估成本** — 支出趋势
- **失败的护栏** — 护栏违规随时间变化
- **线程数量** — 线程数量趋势
- **线程持续时间** — 线程性能趋势
- **线程反馈分数** — 线程的质量指标随时间变化

**配置选项：**

- **项目**：选择要拉取数据的项目
- **指标类型**：从上述任何指标中选择
- **图表类型**：折线图（最适合趋势）或柱状图（适合数量/周期比较）
- **分组**：可选按字段分组数据以查看每组的模式。可用字段取决于数据源：
  - Trace 指标：标签、名称、有错误、错误类型、元数据键
  - Span 指标：标签、名称、有错误、错误类型、元数据键、模型、提供商、Span 类型
  - 线程指标：标签

  当分组激活时，使用**聚合切换**控制数据如何分桶：**总计**显示整个日期范围内每组一个值，而**基于时间**在时间桶中显示值（每小时、每天或每周）。点击图表图例中的标签可直接导航到按该组过滤的 trace 列表。
- **过滤器**：应用 trace 或线程过滤器以专注于基于标签、元数据或其他属性的特定数据
- **反馈分数**：使用反馈分数指标时，可选选择要显示的特定分数（留空显示全部）

### 单一指标组件（多项目）

以紧凑的卡片显示展示单个指标值。适合摘要仪表板和关键绩效指标。

**数据来源：** Trace 或 Span

**Trace 特定指标：**

- 总 trace 计数
- 总线程计数
- 平均 LLM span 计数
- 平均 span 计数
- 每个 trace 的平均预估成本
- 总护栏失败计数

**Span 特定指标：**

- 总 span 计数
- 每个 span 的平均预估成本

**共享指标（trace 和 span 都可用）：**

- P50 持续时间 — 中位持续时间
- P90 持续时间 — 第 90 百分位持续时间
- P99 持续时间 — 第 99 百分位持续时间
- 总输入计数
- 总输出计数
- 总元数据计数
- 平均标签数
- 总预估成本总和
- 输出 token（平均）
- 输入 token（平均）
- 总 token（平均）
- 总错误计数
- 平均反馈分数 — 项目中定义的任何反馈分数

### 指标组件（实验）

比较多个实验的反馈分数。适合可视化 A/B 测试结果和 prompt 迭代结果。

**图表类型：**

- **折线图** — 显示跨实验的趋势（默认）
- **柱状图** — 并排查看详细的分数分布
- **雷达图** — 在径向视图中比较多个实验的反馈分数

**配置选项：**

- **过滤器**：按以下条件过滤实验：
  - 数据集 — 仅显示特定数据集的实验
  - 配置 — 按元数据键和值过滤（例如 model="gpt-4"）
  - 实验 ID — 按 ID 包含特定实验
- **分组**（可折叠，默认折叠）：按以下条件分组聚合结果：
  - 数据集 — 比较不同数据集的结果
  - 配置 — 按元数据键分组以聚合反馈分数（例如按模型类型分组）
  - 支持最多 5 个分组级别用于层次比较
- **最大实验数**：限制显示的实验数量
- **图表类型**：选择折线图、柱状图或雷达图可视化
- **指标**：可选仅显示特定反馈分数（留空显示全部）

### 排行榜组件（实验）

显示一个比较实验的表格，列可配置。适合按特定指标排名实验并一目了然地比较结果。

**配置选项：**

- **过滤器**：与指标组件相同的过滤选项（数据集、配置、实验 ID）
- **分组**：与指标组件相同的分组选项
- **最大实验数**：限制显示的实验数量
- **列**：选择和重新排列要显示的列。列菜单显示所有可用列，带有"N of N 已选"指示器和拖拽手柄用于重新排序
- **排名**：按特定指标排名实验。选项为"无排名"（默认）和任何可用的反馈分数指标。选择"无排名"时，排名顺序选项被禁用

### Markdown 文本组件

适用于多项目和实验仪表板。使用 Markdown 格式添加自定义注释、描述或文档。使用此组件：

- 添加部分标题和说明
- 记录仪表板目的和上下文
- 包含相关资源链接
- 添加团队注释或指南

## 创建工作区仪表板

1. 从侧边栏导航到**仪表板**页面
2. 点击**创建新仪表板**
3. 选择仪表板类型：**多项目**或**实验**
4. 输入名称（描述可选）
5. 点击**创建**

<Frame>
  <img src="/img/production/dashboard_create_dialog.png" alt="创建仪表板对话框" />
</Frame>

## 添加和配置组件

当你点击某个部分中的 **+** 按钮时，会打开一个统一的组件配置模态框：

1. **选择组件类型** — 从顶部的可点击卡片中选择。可用类型取决于仪表板类型：
   - **多项目**：时间序列、单一指标、Markdown
   - **实验**：指标、排行榜、Markdown
2. 在下方配置组件设置。配置区域根据选定的组件类型更新。
3. 每个组件有自己的**项目或实验选择器** — 没有全局仪表板默认值。对于 Insights 视图，当前项目自动设置。
4. 对于图表组件，使用可点击卡片选择**可视化类型**（折线图、柱状图或雷达图）。
5. 点击**保存**添加组件。

## 自定义仪表板

### 添加部分

仪表板由多个部分组成，每个部分包含一个或多个组件：

1. 点击仪表板底部的**添加部分**
2. 给部分命名
3. 向部分添加组件

### 编辑组件

1. 点击任何组件上的菜单图标
2. 选择**编辑**修改组件配置
3. 进行更改并保存

### 重新排列组件

- **拖放**：使用组件上的拖拽手柄在部分内重新排序
- **调整大小**：拖动组件边缘调整大小

### 折叠部分

点击部分标题折叠或展开它。折叠状态在会话间保持。

## 日期范围过滤

使用工具栏中的日期选择器按时间范围过滤数据。选择预设范围（最近 24 小时、最近 7 天等）或选择自定义日期。

**使用日期范围过滤的组件：**

- 时间序列组件 — 将时间序列数据过滤到选定范围
- 单一指标组件 — 在选定范围内计算统计数据

**不受日期范围影响的组件：**

- 实验指标组件 — 无论日期如何都显示实验结果
- 排行榜组件 — 无论日期如何都显示实验结果
- Markdown 文本组件 — 静态内容

## 保存更改

所有仪表板更改**自动保存**。内置 Insights 视图是只读的 — 复制它们以创建可编辑副本。

## 分享仪表板

要分享你当前的仪表板视图：

1. 点击工具栏中的**分享**按钮
2. URL 将复制到你的剪贴板
3. 与有权访问工作区的团队成员分享此 URL

分享的 URL 包含仪表板 ID、活动日期范围和任何活动过滤器，因此接收者看到相同的视图。
