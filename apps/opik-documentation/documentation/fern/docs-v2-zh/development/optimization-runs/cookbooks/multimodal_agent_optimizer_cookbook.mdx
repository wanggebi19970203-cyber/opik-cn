## 什么是多模态优化器示例？

SDK 包含一个完整示例，用于在驾驶危险数据集上优化视觉智能体。它演示了如何通过 `ChatPrompt` 传递图像内容部分、对输出进行评分，以及在 Optimization Studio 中比较试验结果。

## 为什么在此处使用优化器？

多模态提示词对措辞和输出结构非常敏感。运行 HRPO 或 MetaPrompt 可帮助您收敛到更安全、更一致的输出，而无需手动重写提示词。

## SDK 示例的工作原理

1. `multimodal_example.py` 加载驾驶危险数据集（图像 + 危险标签）。
2. 多模态 `ChatPrompt` 在文本指令旁边插入图像 URL 内容部分。
3. 指标（Levenshtein 比率）将预测的危险文本与预期标签进行评分。
4. HRPO 使用训练集优化提示词，并使用小型验证集进行排序。
5. 结果在 Opik UI 中显示（优化运行和试验详情）。
