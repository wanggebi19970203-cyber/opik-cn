## 本示例涵盖的内容

- 从 Opik traces 生成合成问答数据集
- 使用 TinyQA（通过 [tinyqabenchmarkpp](https://pypi.org/project/tinyqabenchmarkpp/)）及其变体如 TinyQA++
- 在合成数据上使用 MetaPrompt 优化提示词
- 在 Opik UI 中查看结果
