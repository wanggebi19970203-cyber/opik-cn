FewShotBayesianOptimizer 是一个精密的提示优化工具，它使用贝叶斯优化技术将您样本问题中的相关示例添加到系统提示中。

<Note>
  当您的主要目标是找到最优的少样本示例（演示）数量和组合来配合您的主指令提示时，`FewShotBayesianOptimizer` 是一个很好的选择，特别适用于**聊天模型**。如果您的任务性能严重依赖于上下文示例的质量和相关性，那么这个优化器是理想的。
</Note>

## 工作原理

`FewShotBayesianOptimizer` 使用贝叶斯优化来找到最优的少样本示例集合和数量，以配合您的基础指令提示用于聊天模型。它使用 [Optuna](https://optuna.org/) 这个超参数优化框架来引导搜索最优的少样本示例集合和数量。

## 快速入门

您可以按照以下步骤使用 `FewShotBayesianOptimizer` 来优化提示：

```python maxLines=1000
from opik_optimizer import FewShotBayesianOptimizer
from opik.evaluation.metrics import LevenshteinRatio
from opik_optimizer import datasets, ChatPrompt

# 初始化优化器
optimizer = FewShotBayesianOptimizer(
    model="openai/gpt-4",
    model_parameters={
        "temperature": 0.1,
        "max_tokens": 5000
    },
)

# 准备数据集
dataset = datasets.hotpot(count=300)

# 定义指标和提示（更多选项请参阅文档）
def levenshtein_ratio(dataset_item, llm_output):
    return LevenshteinRatio().score(reference=dataset_item["answer"], output=llm_output)

prompt = ChatPrompt(
    messages=[
        {"role": "system", "content": "Provide an answer to the question."},
        {"role": "user", "content": "{question}"}
    ]
)

# 运行优化
results = optimizer.optimize_prompt(
    prompt=prompt,
    dataset=dataset,
    metric=levenshtein_ratio,
    n_samples=100
)

# 访问结果
results.display()
```

## 配置选项

### 优化器参数

优化器具有以下参数：

<ParamField path="model" type="str" optional={true} default="openai/gpt-5-nano">用于优化器内部推理（生成少样本模板）的 LiteLLM 模型名称</ParamField>
<ParamField path="model_parameters" type="dict[str, typing.Any] | None" optional={true}>用于优化器内部 LLM 调用的可选 LiteLLM 参数字典。常用参数：temperature、max_tokens、max_completion_tokens、top_p。</ParamField>
<ParamField path="min_examples" type="int" optional={true} default="2">提示中包含的最少示例数量</ParamField>
<ParamField path="max_examples" type="int" optional={true} default="8">提示中包含的最多示例数量</ParamField>
<ParamField path="n_threads" type="int" optional={true} default="8">并行评估的线程数</ParamField>
<ParamField path="verbose" type="int" optional={true} default="1">控制内部日志/进度条（0=关闭，1=开启）</ParamField>
<ParamField path="seed" type="int" optional={true} default="42">用于可重现性的随机种子</ParamField>

### `optimize_prompt` 参数

`optimize_prompt` 方法具有以下参数：

<ParamField path="prompt" type="ChatPrompt">要优化的提示</ParamField>
<ParamField path="dataset" type="Dataset">用于优化的 Opik 数据集</ParamField>
<ParamField path="metric" type="Callable">用于评估的指标函数</ParamField>

### 模型支持

使用 `FewShotBayesianOptimizer` 时需要考虑两个模型：
- `FewShotBayesianOptimizer.model`：用于生成少样本模板和占位符的模型。
- `ChatPrompt.model`：用于评估提示的模型。

`model` 参数接受任何 LiteLLM 支持的模型字符串（例如 `"gpt-4o"`、`"azure/gpt-4"`、`"anthropic/claude-3-opus"`、`"gemini/gemini-1.5-pro"`）。您还可以使用 `model_parameters` 参数传递额外的模型参数：

```python
optimizer = FewShotBayesianOptimizer(
    model="anthropic/claude-3-opus-20240229",
    model_parameters={
        "temperature": 0.7,
        "max_tokens": 4096
    }
)
```
