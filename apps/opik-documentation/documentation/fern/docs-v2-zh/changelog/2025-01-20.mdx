**Opik 控制面板**：

- 为在线评估规则添加了日志，使您更容易确保在线评估指标按预期工作
- 在在线评估规则弹窗的变量映射部分新增自动补全支持
- 在 playground 中新增对 Anthropic 模型的支持
- 在 playground 中使用数据集时现在会创建实验
- 改进了 Opik 主页
- 更新了快速入门中的代码片段，使其更易于理解

**SDK**：

- 改进了对 litellm completion kwargs 的支持
- 放宽了 LiteLLM 的必需版本要求，以避免与其他 Python 包冲突
