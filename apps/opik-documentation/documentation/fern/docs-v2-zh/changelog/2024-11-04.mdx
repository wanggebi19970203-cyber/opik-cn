**SDK**：

- 在 SDK 中引入 `Prompt` 对象，用于管理存储在提示词库中的提示词。更多信息请参阅[提示词管理](/v1/prompt_engineering/prompt_management)指南。
- 引入 `Opik.search_spans` 方法用于搜索项目中的 span。更多信息请参阅[搜索 span](/v1/tracing/export_data#exporting-spans)指南。
- 发布了与 [AWS Bedrock](/v1/integrations/bedrock) 的新集成，用于将 Opik 与 Bedrock 模型配合使用。
