**SDK**：

- 数据集现在支持更灵活的数据模式，我们支持插入任意键值对的条目：

  ```python wordWrap
  import opik

  client = opik.Opik()
  dataset = client.get_or_create_dataset(name="Demo Dataset")
  dataset.insert([
      {
          "user_question": "Hello, what can you do ?",
          "expected_output": {
              "assistant_answer": "I am a chatbot assistant that can answer questions and help you with your queries!"
          }
      },
      {
          "user_question": "What is the capital of France?",
          "expected_output": {
              "assistant_answer": "Paris"
          }
      },
  ])
  ```

- 基于 LiteLLM 集成发布了 WatsonX、Gemini 和 Groq 集成。
- [Hallucination](/v1/integrations/overview) 指标中的 `context` 字段现在是可选的。
- LLM as a Judge 指标现在支持通过指定 `model` 参数来自定义 LLM 提供商。更多信息请参阅[自定义 LLM as a Judge 指标](/v1/evaluation/metrics/overview#customizing-llm-as-a-judge-metrics)部分。
- 修复使用 `update_current_span` 和 `update_current_trace` 方法更新反馈评分时的问题。详情请参阅此 Github issue。
