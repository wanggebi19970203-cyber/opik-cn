以下是自上次发布以来我们做出的最相关的改进：

## 数据集改进

我们通过几项关键改进增强了数据集功能：

- **编辑数据集项** - 你现在可以直接从 UI 编辑数据集项，使更新和优化评估数据更加容易。

- **移除自托管的数据集上传限制** - 自托管部署不再有数据集上传限制，为大规模评估提供更大灵活性。

- **数据集项标签支持** - 添加了数据集项的全面标签支持，改善评估数据的组织和过滤。

- **按任意列过滤数据集** - 在 Playground 和数据集视图中按任意列过滤数据集，提供灵活的方式来查找和使用特定数据子集。

- **重命名数据集** - 直接从 UI 重命名数据集，便于组织和管理评估数据集。

## 实验更新

我们对实验管理和分析进行了重大改进：

- **实验级指标** - 计算实验级指标（与实验项级指标相对），更好地洞察评估结果。在[实验级指标文档](/v1/evaluation/evaluate_your_llm#computing-experiment-level-metrics)中了解更多。

- **重命名实验和元数据** - 直接从仪表板更新实验名称和元数据配置，为你提供对实验组织的更多控制。

- **Token 和成本列** - Token 使用量和成本现在在实验项表中展示，便于快速扫描和成本可见性。

## Playground 改进

我们使 Playground 更加强大，对非技术用户更易使用：

- **从 Playground 轻松导航到数据集和指标** - 从 Playground 到相关数据集和指标的快速导航链接，简化你的工作流。

- **Playground 数据集的高级过滤** - 按标签和任何其他列过滤 Playground 数据集，使查找和使用特定数据集项更加容易。

- **Playground 分页** - 添加了分页支持，更高效地处理 Playground 中的大型数据集。

- **在 Playground 中添加实验进度条** - 运行中实验的可视化进度指示器，实时反馈实验状态。

- **在 Playground 中添加模型特定的限流和并发配置** - 在 Playground 中按模型配置限流和并发设置，提供对资源使用的细粒度控制。

## 增强的告警

我们通过阈值支持扩展了告警功能：

- **为 Trace 和对话线程反馈评分添加阈值支持** - 为 traces 和对话线程的反馈评分配置阈值，实现基于质量指标的更精确告警。

- **为 Trace 错误告警添加阈值** - 设置 trace 错误告警的阈值，仅在错误率超过配置限制时收到通知。

- **从 Playground 触发实验创建告警** - 当从 Playground 创建实验时接收告警。

## Opik 优化器更新

Opik 优化器的重大增强：

- **成本和延迟优化支持** - 添加了同时优化成本和延迟指标的支持。在[优化指标文档](/v1/agent_optimization/optimization/define_metrics#include-cost-and-duration-metrics)中了解更多。

- **训练和验证数据集支持** - 引入了训练和验证数据集拆分支持，实现更好的优化工作流。在[数据集文档](/v1/agent_optimization/optimization/define_datasets#trainvalidation-splits)中了解更多。

- **Microsoft Agents 和 CrewAI 的示例脚本** - 新的示例脚本演示如何将 Opik 优化器与流行的 LLM 框架一起使用。查看[示例脚本](https://github.com/comet-ml/opik/tree/main/sdks/opik_optimizer/scripts/llm_frameworks)。

- **UI 增强和优化器改进** - 多项 UI 增强以及 Few Shot、MetaPrompt 和 GEPA 优化器的各种改进，提升可用性和性能。

## 用户体验增强

改善了平台的整体可用性：

- **添加 `has_tool_spans` 字段以在对话线程视图中显示工具调用** - 工具调用现在在对话线程视图中可见，提供更好的 Agent 工具使用可见性。

- **直接从 Trace、对话线程和 Span 详情视图添加导出功能（JSON/CSV）** - 直接从详情视图以 JSON 或 CSV 格式导出数据，便于分析和分享可观测性数据。

## 新模型！

扩展了模型支持：

- **添加了 Gemini 3 Pro、GPT 5.1、OpenRouter 模型支持** - 添加了对最新模型版本的支持，包括 Gemini 3 Pro、GPT 5.1 和 OpenRouter 模型，让你访问最新的 AI 能力。
