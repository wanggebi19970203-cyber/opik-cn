## 高级过滤和搜索功能

我们扩展了过滤和搜索功能，帮助你更有效地查找和分析数据：

- **自定义 Trace 过滤器**：支持对 traces 和 spans 的输入/输出字段进行自定义过滤，实现更精确的数据筛选
- **增强的搜索**：改进了搜索功能，具有更好的结果高亮和代码块中的本地搜索
- **更好的搜索结果**：增强了搜索结果高亮和代码块中的本地搜索功能
- **崩溃过滤**：修复了包含 `%` 等特殊字符的值的过滤问题，防止崩溃
- **数据集过滤**：添加了按 datasetId 和 promptId 过滤实验的支持

## 指标和分析改进

我们增强了指标和分析功能：

- **对话线程反馈评分**：添加了全面的对话线程反馈评分系统，更好地评估对话质量
- **对话线程时长监控**：指标仪表板中的新时长组件，用于监控对话长度趋势
- **在线评估规则**：添加了启用/禁用在线评估规则的功能，实现更灵活的监控
- **成本优化**：减少了成本 prompt 查询，提高性能并减少不必要的 API 调用

## 用户体验增强

我们进行了多项用户体验改进，使平台更加直观和高效：

- **全屏弹窗改进**：增强了全屏弹窗体验，具有更好的导航和可用性
- **标签组件优化**：使标签组件更小更紧凑，提高空间利用率
- **列排序**：在所有 Prompt 列上启用排序和过滤，改善数据组织
- **多项目标签**：添加了在 Traces 和 Spans 表中同时为多个项目添加标签的功能

## SDK、集成和文档

- **LangChain 集成**：增强了 LangChain 集成，改进了 provider 和 model 日志记录
- **Google ADK 集成**：更新了 Google ADK 集成，具有更好的图表构建功能
- **Bedrock 集成**：为 ChatBedrock 和 ChatBedrockConverse 添加了全面的成本追踪支持

## 安全和稳定性增强

我们实施了多项安全和稳定性改进：

- **依赖更新**：更新了关键依赖，包括 MySQL connector、OpenTelemetry 和各种安全补丁
- **错误处理**：改进了平台的错误处理和日志记录
- **性能监控**：增强了 NewRelic 支持以改善性能监控
- **Sentry 集成**：在 Sentry 事件中添加了更多关于包版本的元数据，以便更好地调试
