## 产品增强
- 支持通过用户界面直接上传 **CSV 数据集**
- 在实验表中添加**实验成本追踪**
- 在平台各处为**新用户引导**添加提示和帮助
- 在 traces 表中添加了"LLM 调用次数"
- 复杂 Agent 对话线程的美化格式
- 前端预览 **MP3** 文件的支持

## SDK 和 API 增强
- JS 开发者的好消息！我们发布了 **JS SDK 的实验支持**（官方文档即将推出）
- 新的实验批量 API：引入了用于批量记录实验的新 API
- API 和 SDK 的速率限制改进

## 集成
- Playground 中添加了对 OpenAI o3-mini 和 Groq 模型的支持
- OpenAI Agents：实现了上下文感知并提升了健壮性。改进了对话线程处理
- Google ADK：添加了多 Agent 集成支持
- LiteLLM：为 SDK 调用添加了 token 和成本追踪。集成现在兼容 opik.configure(...)
