## 新功能

- **Opik Agent 优化器**：一套综合性工具包，旨在提升大语言模型（LLM）应用的性能和效率。[了解更多](/v1/agent_optimization/overview)

- **Opik 护栏**：护栏帮助你保护应用免受 LLM 固有风险的影响。用于检查 LLM 调用的输入和输出，检测离题回答或敏感信息泄露等问题。[了解更多](/v1/production/guardrails)

## 产品增强

- **Playground 中的新 Prompt 选择器** — 从 Prompt 库中选择已有 prompts，简化测试工作流。
- **Agent 的"美化格式"改进** — 提升了 UI 中复杂对话线程的可读性。

## 集成

- **Vertex AI (Gemini)** — 直接在 Opik 中集成离线和在线评估支持。现已在 Playground 中可用。
- **JS/TS SDK 中的 OpenAI 集成**
- **AWS Strands Agents**
- **Agno 框架**
- **Google ADK 多 Agent 支持**

## SDK 和 API 增强

- **OpenAI LLM 高级配置** — 支持自定义 headers 和 base URLs。
- **Span 时间精度** — 时间分辨率提升至微秒级别，实现更精确的监控。
- **更好的错误信息** — SDK 验证和运行时错误提供了更具描述性的错误信息。
- **基于流的追踪和增强的流式支持**
